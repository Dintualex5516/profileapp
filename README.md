# Simple Flutter Profile App



A basic Flutter application task that displays a user's profile.

## Folder Structure

 `screens/`: main screens - Home and Profile screens
 `models/`: User model class
`widgets/`: custom  widgets which can reuse
  images- Does not contain any assets folder loaded a static network image 


# features of this app
 Two screens: home and profile
Navigation using `Navigator.push` and `Navigator.pop`
 circular profile image and user info cards..
 Reusable widget: `ProfileInfoCard`
loading indicator before profile loads
 


# to run this app 

first Clone the repo
then Run `flutter pub get`
after  Use `flutter run` to launch the app .
thats it...
